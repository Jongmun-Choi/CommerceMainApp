# CommerceMainApp

1. 사용 Library
- Jetpack Compose(UI 개발)
- Jetpack Paging(데이터 로딩 Support)
- Jetpack DataSource(데이터 저장용)
- Retrofit(네트워크)
- Splash(UI 개발)
- Glide(이미지 로딩)
- Hilt(Di)
- Moshi(Json Converter)
2. 구조 : Mvvm
3. 기능
  - MockServer를 통한 데이터 조회
    - Section, Product List 조회
  - Section Type 별 다른 형태의 Product Card Ui제공
  - 찜등록 기능
  - Paging을 통한 데이터 로딩
  - Custom Splash
